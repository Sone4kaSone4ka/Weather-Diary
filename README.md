# Weather-Diary

Автор: [Чебыкина Софья Сергеевна]

Описание программы
«Дневник погоды» — это простое и удобное графическое приложение, написанное на языке Python. Оно позволяет вести учёт погодных наблюдений: добавлять записи о температуре, описании погоды и осадках за определённую дату. Все данные сохраняются в файл формата JSON, что обеспечивает их сохранность между запусками программы.

Приложение поддерживает фильтрацию записей по дате и температуре, а также позволяет быстро сбросить все фильтры.

Технологии
Язык программирования: Python 3.x
Библиотека для GUI: Tkinter (входит в стандартную библиотеку Python)
Формат хранения данных: JSON
Системные требования
Операционная система: Windows, macOS или Linux.
Установленный интерпретатор Python версии 3.6 или выше.
Как запустить приложение
Установка Python:Если у вас не установлен Python, скачайте его с официального сайта python.org и установите. Убедитесь, что при установке стоит галочка «Add Python to PATH».
Подготовка проекта:Скачайте или клонируйте весь проект на свой компьютер.
Запуск:Откройте терминал (командную строку) в папке с проектом и выполните команду:
bash
Копировать
python weather_diary.py

Использование программы
1. Добавление новой записи
В поле Дата введите дату в формате ДД.ММ.ГГГГ (например, 05.05.2026).
В поле Температура введите числовое значение (например, 22.5).
В поле Описание кратко опишите погоду (например, Ясно, Пасмурно, Сильный ветер).
Поставьте галочку Осадки, если в этот день были осадки.
Нажмите кнопку Добавить запись.
2. Просмотр записей
Все добавленные записи отображаются в таблице ниже формы ввода.

3. Фильтрация записей
По дате: Введите конкретную дату в формате ДД.ММ.ГГГГ и нажмите «Применить фильтр», чтобы увидеть записи только за этот день.
По температуре: Введите числовое значение (например, 10) и нажмите «Применить фильтр», чтобы увидеть записи, где температура была выше указанного значения.
4. Сброс фильтров
Чтобы снова увидеть все сохраненные записи, нажмите кнопку Показать все.

Как работает сохранение данных
Приложение автоматически сохраняет все записи в файл weather_data.json при добавлении новой записи. При запуске программа загружает данные из этого файла. Вам не нужно беспокоиться о сохранении вручную.
